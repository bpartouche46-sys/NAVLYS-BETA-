# Publer · Setup NAVLYS (FR + EN)

---

## FR — Procédure express

1. Créer un compte Publer (plan Business) sur https://publer.io
2. Connecter ces réseaux dans l'ordre :
   - LinkedIn (page perso + page entreprise NAVLYS)
   - X / Twitter
   - Instagram (compte business)
   - Facebook Page
   - TikTok
   - YouTube
   - Threads
   - Pinterest
   - Mastodon
   - Bluesky
3. Créer la **bibliothèque NAVLYS** :
   - Library → Folder « NAVLYS Brand Kit »
   - Importer les bios FR + EN (texte)
   - Importer les visuels (à venir, dossier `/visuels/`)
4. Créer le **calendrier 90 jours** :
   - 3 posts/semaine (mardi 9 h · jeudi 12 h · samedi 10 h, heure de Paris)
   - Rotation : 1 question publique · 1 micro-analyse · 1 storytelling NAVLYS
5. Brancher Workspace OneDrive Downloads comme source d'assets (auto-import).
6. Activer les rapports hebdomadaires par email (lundi 8 h).

---

## EN — Express procedure

1. Create a Publer account (Business plan) at https://publer.io
2. Connect these networks in order:
   - LinkedIn (personal + NAVLYS company page)
   - X / Twitter
   - Instagram (business account)
   - Facebook Page
   - TikTok
   - YouTube
   - Threads
   - Pinterest
   - Mastodon
   - Bluesky
3. Build the **NAVLYS library**:
   - Library → Folder "NAVLYS Brand Kit"
   - Import FR + EN bios (text)
   - Import visuals (coming, `/visuels/`)
4. Build the **90-day calendar**:
   - 3 posts/week (Tue 9 a.m. · Thu 12 p.m. · Sat 10 a.m., Paris time)
   - Rotation: 1 public question · 1 micro-analysis · 1 NAVLYS storytelling
5. Wire the Workspace OneDrive Downloads as asset source (auto-import).
6. Enable weekly email reports (Monday 8 a.m.).
