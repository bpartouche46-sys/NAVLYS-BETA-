## 🇫🇷 Mettre en ligne en 3 minutes (Vercel)

1. Va sur `https://vercel.com` → connecte-toi (compte déjà créé).
2. **Add New → Project → Deploy**.
3. Choisis **"Deploy Static Site"** (ou "Import third-party / upload").
4. **Glisse-dépose le dossier `NAVLYS_VOILE_TEASER/`** entier (avec `index.html` + `vercel.json`).
5. Nom du projet : `navlys-voile` → **Deploy**.
6. En ~30 s, tu as une URL `https://navlys-voile-xxxx.vercel.app` : **c'est en ligne**, testable sur mobile et ordi.

### Pointer le vrai domaine
7. Projet Vercel → **Settings → Domains → Add** → `navlys.com` (et `www.navlys.com`).
8. Suis ton pack `NAVLYS_DNS_BASCULE_PACK/` pour la bascule DNS (sans couper l'email `bruno@navlys.com`).

### Mettre la voile sur CHAQUE site
- Pour chaque domaine/site : déploie la même `index.html` comme page d'accueil, **ou** ajoute le domaine au projet `navlys-voile`.
- Au lancement (31 mai), tu remplaces `index.html` par le vrai site (`navlys/` ou `NAVLYS_PORTAL_APP/`). La voile tombe, le port s'ouvre.

## 🇬🇧 Go live in 3 minutes (Vercel)

1. Go to `https://vercel.com` → log in (account already created).
2. **Add New → Project → Deploy**.
3. Choose **"Deploy Static Site"** (or "Import third-party / upload").
4. **Drag-and-drop the whole `NAVLYS_VOILE_TEASER/` folder** (with `index.html` + `vercel.json`).
5. Project name: `navlys-voile` → **Deploy**.
6. In ~30s you get a `https://navlys-voile-xxxx.vercel.app` URL: **it's live**, testable on mobile and desktop.

### Point the real domain
7. Vercel project → **Settings → Domains → Add** → `navlys.com` (and `www.navlys.com`).
8. Follow your `NAVLYS_DNS_BASCULE_PACK/` for the DNS switch (without breaking the `bruno@navlys.com` email).

### Put the veil on EVERY site
- For each domain/site: deploy the same `index.html` as the homepage, **or** add the domain to the `navlys-voile` project.
- At launch (31 May), replace `index.html` with the real site (`navlys/` or `NAVLYS_PORTAL_APP/`). The veil drops, the harbor opens.
