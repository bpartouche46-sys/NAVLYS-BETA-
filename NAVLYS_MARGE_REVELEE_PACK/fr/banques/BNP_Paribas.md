# Fiche banque · BNP Paribas (FR)

## Marge moyenne par client retail
- ~ 380 € / an pour un client multi-produits (compte + livret + assurance vie + crédit)
- Sources : rapports annuels BNP Paribas + analyses sectorielles 2024-2025

## Produits opaques
- Assurance vie multi-supports BNP Cardif : frais de gestion UC ~ 1,0 % + frais sur versement
- Fonds maison surreprésentés dans les profils gérés sous mandat
- Crédit immobilier : marge d'arbitrage + assurance emprunteur groupe

## Axes d'amélioration côté client
1. Renégocier l'assurance emprunteur (loi Lemoine)
2. Externaliser l'assurance vie chez un acteur indépendant
3. Auditer les frais de tenue de compte et services payants inutilisés

## Contre-stratégie NAVLYS
- Audit privé 360° : identifier les 3 plus gros postes de fuite
- Bascule produits opaques → architecture ouverte
- Tableau de bord patrimonial trimestriel
