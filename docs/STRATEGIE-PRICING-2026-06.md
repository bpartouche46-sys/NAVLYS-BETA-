## 0. Les 5 principes « best-in-class » (communs aux 3 produits)
1. **Freemium = obligatoire** (norme du secteur) → un palier gratuit comme aimant d'acquisition.
2. **Mensuel + annuel remisé** (~2 mois offerts) + **essai gratuit** → conversion & rétention.
3. **Palier premium = decoy** : il existe surtout pour valoriser le palier du milieu (le best-seller).
4. **Modèle différenciant** : là où les concurrents imposent l'abo, NAVLYS peut casser le code (ex. NAVBIO en **paiement unique**).
5. **La conformité est un argument de VENTE** : « zéro CIF/ORIAS », « ne remplace pas un avocat » = gage de confiance (cf. sanction FTC contre DoNotPay, janv. 2025), pas une faiblesse.
