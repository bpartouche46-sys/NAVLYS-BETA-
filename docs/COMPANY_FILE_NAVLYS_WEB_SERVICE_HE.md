## 5. יעד המס — למה התקנון מנוסח "SaaS / היי-טק"

התקנון (בקובץ נפרד) ממצב את החברה כ**מפעל טכנולוגי**, כרטיס הכניסה
למסלולי המס המועדפים לפי החוק לעידוד השקעות הון:

| מסלול | מס חברות | תנאים עיקריים |
|---|---|---|
| **מפעל טכנולוגי מועדף (PTE)** | **12 %** (במקום 23 %) · **7.5 %** באזור פיתוח א' | מו"פ ≥ 7 % מהמחזור (ממוצע 3 שנים) או מו"פ מעל 75 מיליון ש"ח; ובנוסף אחד מאלה: ≥ 20 % מהעובדים במו"פ, השקעת הון סיכון ≥ 8 מיליון ש"ח, צמיחה ≥ 25 % בשנה |
| מפעל מועדף (תעשייתי, כולל תוכנה) | 16 % · 7.5 % אזור א' | ייצור/תוכנה, ללא מבחן המו"פ המלא |
| דיבידנד מרווחי PTE | ניכוי במקור 20 % | — |
| מענקי מו"פ | רשות החדשנות | תיק פרויקט |

**אמת חשובה:** ההטבות אינן נובעות מהתקנון עצמו — התקנון פותח את הדלת,
אך הזכאות מוכחת בפעילות בפועל ובספרים: מעקב נפרד אחר עלויות מו"פ
(יעד ≥ 7 % מהמחזור), קניין רוחני בבעלות החברה, הכנסות מזכות
(מנויי SaaS ורישיונות) בחשבוניות החברה, תיעוד כוח אדם במו"פ, ובקשה
שנתית לרשות המסים (אפשרות: פרה-רולינג). צירוף אשקלון לאזור פיתוח א'
(7.5 %) מקודם בחקיקה — לאישור רואה החשבון ביום ההגשה.

**השאלה לרואה החשבון, מילה במילה:** "אני מקים חברת SaaS/מו"פ — אני
רוצה מעמד של **מפעל טכנולוגי מועדף** (12 %, או 7.5 % אם אזור א' חל על
היישוב שלי): אנא אשרו את מטרות החברה שבתקנון והקימו מעקב עלויות מו"פ
בשיעור 7 % לפחות מהמחזור."
