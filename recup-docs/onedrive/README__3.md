# BRUNO COIN V2 — Kit Réseaux Sociaux

Visuels HTML/SVG prêts à exporter pour LinkedIn, X, Instagram, Discord, YouTube.

## Mode d'emploi rapide

1. Ouvre le fichier HTML correspondant à ta cible dans Chrome
2. F12 → toolbar Device → Responsive → règle aux dimensions exactes
3. Clic droit dans la page → « Capture d'écran » → enregistre le PNG
4. Importe le PNG dans le réseau social ciblé
5. Récupère la caption depuis `../_BRUNO_COIN_v2_KIT_RESEAUX.md`

## Pour exporter une rotation animée (GIF)

Outils recommandés :
- **Mac** : Kap (open source, gratuit) → enregistre la zone du coin pendant 10s
- **Windows** : ScreenToGif (gratuit) → idem
- **Linux** : Peek

Configuration suggérée : 30 fps, durée 10s (1 tour complet), résolution sortie 600×600 px (Instagram) ou 800×800 (LinkedIn).
