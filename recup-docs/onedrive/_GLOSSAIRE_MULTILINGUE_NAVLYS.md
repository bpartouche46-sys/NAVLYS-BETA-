## ⚠️ RÈGLES ABSOLUES (toutes langues confondues)

1. **G1 conformité** : aucune traduction ne peut promettre un rendement, garantir un résultat, ou laisser entendre un conseil personnalisé. Si le traducteur natif "embellit" une formule G1 → **rejet immédiat**.
2. **Bruno n'est PAS CIF / PAS ORIAS** : ne JAMAIS traduire par « conseiller en investissement », « cabinet de gestion », « advisor » au sens réglementé, « Anlageberater » (DE = encadré BaFin), « consulente finanziario » (IT = encadré CONSOB), etc. Toujours : **éditeur de contenu pédagogique / educator / publisher / автор обучающего контента**.
3. **Esprit poétique préservé** : les mots `rêve / cap / voyage / horizon / vent` doivent rester poétiques même en allemand ou en russe. Si la traduction littérale est sèche, c'est qu'elle est mauvaise.
4. **Marques déposées non traduites** : NAVLYS, NAVBIO, NAVBIOLIFE, NAVLYS.IO, BRUNO COIN restent identiques dans toutes les langues. Translittération en cyrillique/arabe/CJK acceptée pour prononciation, mais le mot reste en latin sur le site.
