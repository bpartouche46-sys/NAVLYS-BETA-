## 🎯 Règles d'or universelles

1. **Captions toujours** (TikTok pousse les vidéos avec captions visibles ; 85 % des users regardent sans son).
2. **Premier mot accroche** dans les 3 premières secondes — sinon swipe.
3. **CTA URL toujours en dernier** + dans la caption.
4. **Disclaimer overlay petit** mais présent à la dernière seconde.
5. **Hashtag #NAVLYS systématique** + 2-4 hashtags Tier 2 max.
