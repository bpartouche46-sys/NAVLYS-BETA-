## 6. Mode d'emploi — comment utiliser ce kit

1. Ouvre les fichiers HTML du dossier `_KIT_RESEAUX_V2/` dans Chrome
2. Mets-toi en mode Responsive aux dimensions cibles (DevTools F12)
3. Clic droit → « Capturer la zone » ou « Capture d'écran complète »
4. Récupère le PNG, importe dans LinkedIn / X / Instagram
5. Copie-colle la caption correspondante depuis ce document

**Tip** : pour exporter en GIF rotation, utilise un outil type Kap (Mac)
ou ScreenToGif (Windows) pendant que le coin tourne dans le navigateur.
