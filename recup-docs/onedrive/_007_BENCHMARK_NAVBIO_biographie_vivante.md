## 2. Forces / faiblesses du marché vs NAVBIO

### ⚠️ Ce que les concurrents font mieux que NAVBIO aujourd'hui
- **Trafic organique** : Storyworth, MyHeritage dominent SEO US (millions de visites). NAVBIO part de 0.
- **Réseau de témoignages crédibles** : Remento a Shark Tank, Storyworth a 10 ans d'avis Trustpilot.
- **Distribution livre papier intégrée** : presque tous incluent l'impression. NAVBIO doit décider si livre papier = add-on ou inclus.

### 🚀 Ce que NAVBIO peut faire mieux (gaps marché)

1. **One-shot vrai (19/29/39 €)** : aucun concurrent n'a un point d'entrée < 50 €. NAVBIO unique sur cette tranche.
2. **Multimédia natif (vidéo, audio, photo)** : Remento fait audio, mais aucun fait cinéma de montage IA. **NAVBIO Cinéma 149 €** = niche vide.
3. **i18n sérieuse** : tous les concurrents = EN-only ou FR-only. NAVBIO peut être premier multilingue (FR/EN/HE/AR/ES/IT).
4. **Transmission héritiers organisée** : Trust & Will fait juridique pur, GoodTrust fait coffre numérique, mais **aucun ne couple biographie + transmission** notariale Méditerranée. **Gap énorme.**
5. **Pas d'abonnement = pas de friction** : argument anti-MRR fort pour la cible "+50 ans" qui déteste les abonnements (preuve : Trustpilot reviews concurrents).
6. **Accessibilité WCAG** : 0/10 chez tous (interfaces écrasantes). NAVBIO peut viser AAA dès le départ.
