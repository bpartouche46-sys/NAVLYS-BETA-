## 2. Top 10 partenaires prioritaires (ordre business)

### #1 — Trade Republic Partners
- **Programme** : OUI ([traderepublic.com/partners](https://traderepublic.com/partners) — accès via app perso)
- **CPA standard** : 10-30 €/filleul (variable selon pays + funding)
- **Statut compte Bruno** : compte client perso probable (code parrainage évoqué), **affilié pro à ouvrir**
- **Paiement** : virement SEPA mensuel
- **Périodicité** : M+1, palier 25 €
- **Pourquoi #1** : programme le plus actif en EU, KYC rapide, et 0 € commission ordre = aimant prospects
- **Action Bruno J0** : se connecter via app, demander accès Partners

### #2 — Binance Affiliates
- **Programme** : OUI ([binance.com/en/activity/affiliate](https://binance.com/en/activity/affiliate))
- **Commission** : 20-50 % rev share sur les fees des filleuls (à vie)
- **Paiement** : crypto (BTC/USDT) ou SEPA
- **Périodicité** : hebdomadaire
- **Pourquoi #2** : récurrent à vie, palier bas (10 USDT)
- **Action Bruno J-2** : se logger Binance (KYC déjà fait), demander accès Affiliates depuis dashboard

### #3 — eToro Partners
- **Programme** : OUI ([etoropartners.com](https://etoropartners.com/))
- **CPA standard** : $100-250 par compte funded ≥ $200
- **Bonus** : Popular Investor (AUM 1.5 % annuel) — Bruno peut copier-traderé visible
- **Paiement** : virement SWIFT mensuel
- **Périodicité** : M+1, palier $250
- **Action Bruno J0** : formulaire en ligne, validation 1-3 jours

### #4 — TradingView Refer (50 % recurring)
- **Programme** : OUI ([tradingview.com/refer-a-friend](https://www.tradingview.com/refer-a-friend/))
- **Commission** : 50 % des abos premium des filleuls, à vie tant que le filleul reste abonné
- **Paiement** : PayPal ou virement
- **Périodicité** : M+1, palier 100 $
- **Pourquoi prioritaire** : marges récurrentes énormes (50 % de $14.95-$59.95/mois × N filleuls)
- **Action Bruno J-3** : compte gratuit TradingView puis activer Refer-A-Friend (5 min)

### #5 — Wise Business Affiliates (via Awin)
- **Programme** : OUI ([wise.com/affiliate-program](https://wise.com/affiliate-program))

- **Paiement** : virement SEPA via Awin
- **Action Bruno J+7** : créer compte Awin puis postuler programme Wise

### #6 — IBKR Refer-A-Friend
