# 07 — SÉLECTION MUSIQUE EPIDEMIC SOUND
> Trois pistes proposées, classées par adéquation à la vidéo manifeste NAVLYS.
> Plan recommandé : **Personal** (9 $/mois) — licence broadcast monde tous réseaux.

---

## 🎵 PISTE A — RECOMMANDÉE : « Quiet Horizon » (ou équivalent)

### Profil
- **Genre** : Ambient minimal, cinematic.
- **Tempo** : 60 BPM (rythme cardiaque calme).
- **Tonalité** : Do mineur ou Si bémol majeur.
- **Instrumentation** : Pad cordes synthétisées + nappes piano sustainées + très faible sub-bass.
- **Durée native** : 2 min 30 s (donc largement plus que les 30 s nécessaires).

### Pourquoi cette piste
- 60 BPM = rythme méditatif, raconté à voix posée → parfait pour Bruno.
- Pas de percussion → ne « pulse » pas et n'épuise pas l'attention.
- Tonalité mineure douce = registre émotionnel « havre / sérénité ».
- Très peu d'instruments → ne masque jamais la voix de Bruno.

### Pistes alternatives à chercher dans Epidemic Sound
Tags à rechercher : *cinematic ambient + 60 BPM + 1-3 min*. Bonnes alternatives :
- "Slowly Drifting" — Cody Martin
- "Distant Memory" — Marc Torch
- "The Stillness" — Ben Elson

### Plan de mix dans Descript
| Temps | Action |
|---|---|
| **0 → 2 s** | Fade-in 0 → −18 dB (jamais à 0 dB). |
| **2 → 22 s** | Volume constant **−18 dB** (sous la voix Bruno). |
| **22 → 25 s** | Léger boost à **−15 dB** (la voix se tait progressivement). |
| **25 → 28 s** | Pic doux à **−10 dB** pour accompagner l'outro BRUNO COIN. |
| **28 → 30 s** | Fade-out 0.7 s vers silence + cymbale brushée subtile pour clore. |

## 🎵 PISTE B — ALTERNATIVE : « First Light » (montée douce piano)

### Profil
- **Genre** : Piano cinematic, progressive.
- **Tempo** : 72 BPM.
- **Tonalité** : Ré majeur lumineux.
- **Instrumentation** : Piano solo → ajout violoncelle → ajout cordes à 20 s.
- **Durée native** : 1 min 50 s.

### Pourquoi cette piste
- Construction émotionnelle qui monte → idéal pour la **Version B (émotionnelle 32 s)**.
- Tonalité majeure plus optimiste → adapté à un message d'espoir / sérénité construite.

### Inconvénient
- Risque de **« sur-vendre »** l'émotion — la sobriété recommandée par la charte impose la Piste A pour la Version A directe.

### Pistes alternatives à chercher
- "Sunlit Path" — Kara Square
- "New Dawn Above" — Hampus Naeselius
- "The Pact" — Tracktribe

## 🎵 PISTE C — ALTERNATIVE : « Mediterranean Breath » (instrumental méditerranéen)

### Profil
- **Genre** : Acoustic Mediterranean folk / nylon guitar.
- **Tempo** : 78 BPM.
- **Tonalité** : Ré mineur andalou.
- **Instrumentation** : Guitare nylon + cajón soft + accordéon discret.

### Pourquoi cette piste
- Référence directe à l'identité skipper de Bruno + Méditerranée.
- Couleur sonore distinctive — rare dans le paysage des manifests tech.

### Inconvénient
- Peut être perçu comme « niche » ou « régional ». À tester en A/B sur audience FR vs international.

### Pistes alternatives
- "Sirocco" — Reynard Seidel
- "Cassis" — Christoffer Moe Ditlevsen
- "Olive Tree" — Caleb Etheridge

## 🎚️ RÉGLAGES TECHNIQUES POUR LES 3 PISTES

### Format à utiliser
- **WAV 44.1 kHz / 24-bit** (broadcast standard).
- MP3 320 kbps acceptable mais pas optimal pour le master.

### EQ recommandée (à appliquer dans Descript Audio → Equalizer)
- **High-pass filter 80 Hz** : enlève les sub-basses qui rentreraient en conflit avec la voix masculine de Bruno (fondamentale ~110-150 Hz).
- **Notch -6 dB à 250 Hz** : creuse la « boue » qui ferait paraître la voix moins claire.
- **Boost +2 dB à 2 kHz** : augmente la présence musicale sans masquer la diction Bruno.
- **Shelf -3 dB au-dessus de 8 kHz** : enlève l'air aigu qui pourrait fatiguer en écoute mobile.

### Ducking automatique
- Activer **Auto-Duck Audio** dans Descript.
- Threshold : -25 dB.
- Ratio : 4:1.
- Attack : 50 ms.
- Release : 300 ms.
- Effet : dès que la voix Bruno parle, la musique baisse automatiquement de 8 dB.

### Limiter master final
- Ceiling : −1 dB (jamais 0 dB pour éviter le clipping sur réseaux sociaux qui re-compressent).
- LUFS cible : **−14 LUFS** intégré (norme YouTube/Spotify) → **−16 LUFS** pour LinkedIn/Insta (mieux toléré).

## 🆘 PLAN DE SECOURS SI EPIDEMIC SOUND N'EST PAS ACTIVÉ

- **Artlist.io** (16 $/mois) — qualité supérieure, gros catalogue.

- **Free Music Archive (FMA)** — sous licence CC0 ou CC-BY.
- **Pixabay Music** — gratuit, broadcast OK. Cherche tags : *ambient cinematic 60 BPM*.
